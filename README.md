# SpaceX Falcon 9 Landing Prediction 

## Project Overview
This project predicts whether the Falcon 9 first stage will successfully land using machine learning techniques.

## Data Sources
- SpaceX API
- Web scraping (Wikipedia)

## Methods Used
- Data Wrangling
- Exploratory Data Analysis (EDA)
- SQL Analysis
- Interactive Visualization (Folium & Plotly Dash)
- Machine Learning Models (LogReg, SVM, KNN, Decision Tree)

## Results
- Best Accuracy: ~83%
- Key Factors:
  - Flight Number
  - Orbit Type
  - Payload Mass

## Tools & Technologies
- Python (Pandas, NumPy, Scikit-learn)
- SQL (SQLite)
- Plotly Dash
- Folium

## Conclusion
Landing success can be predicted with good accuracy, helping estimate launch cost efficiency.
